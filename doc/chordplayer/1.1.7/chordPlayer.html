Class: chordPlayer

chordPlayer

A musical chord that can be played in a browser. Plays for however long it has been defined to.

Constructor

new chordPlayer(chordName, audioContextopt)

Parameters:
Name Type Attributes Description
chordName string name of the chord, i.e "Cmin7b5"
audioContext Object <optional>
 WebAudio audioContext
Properties:
Name Type Attributes Description
octave number musical octave of the note forming the chord [1-8]
name string Full name of the chord
duration number length of time the sound has to be played, in seconds
volume number volume [0-1]
notes Array Array of notePlayer instances, these are the notes composing the chord
isPlaying Bool true if the sound is being played
verbose Boolean verbose mode (true/false)
audioContext Object <optional>
 WebAudio audioContext. Created if not provided
destinationNode Object WebAudio destinationNode
